# AeroNav

# Prerequisites
Python 3.*
Flask 2.*
Mongo DB Local Server
            
# How to run

## Run Mongo DB Server
1. run `mongos` cmd on one terminal
2. run `mongo` cmd on 2nd terminal
3. run `Flask run` cmd on 3rd terminal
4. search `{url displayed on 3rd terminal}/populateDB` to add sample data to DB
